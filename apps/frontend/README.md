# Frontend Module

This module contains additional UI components for managing agent configurations and logs.
It complements the existing `apps/renderer` Vue application.

## Setup

From repository root:
```bash
npm install
npm run dev
```
The frontend is served via Vite during development.

## Structure
```text
apps/frontend/
  components/        # Vue components
  hooks/             # composables for state management
  composables/       # utility functions
  pages/             # view components
  styles/            # Tailwind configuration
```
